Flutter 在 Windows 的环境搭建

Posted on 2018-08-19 | In Flutter

安装 Flutter

下载 Futter

运行如下命令，从 GitHub 下载 Flutter:

$ git clone https://github.com/flutter/flutter.git

配置环境变量

  • Path 条目下的配置

    “控制面板>用户帐户>用户帐户>更改我的环境变量”。在“用户变量”下“Path”条目:添加 flutter\bin 的全路径。

  • 添加条目

    PUB_HOSTED_URL https://pub.flutter-io.cn
    FLUTTER_STORAGE_BASE_URL https://storage.flutter-io.cn

添加依赖

执行如下命令：
$ flutter doctor

成功之后，如果提示 flutter 版本太旧需要更新，就执行命令 $ flutter upgrade 更新。

使用 Android Studio 创建 Flutter 项目

配置 Android Studio

在 Android Studio 欢迎页右下角，点击 Configure ，选择 Settings > Plugins > Browse repositories , 搜索 flutter 插件并安装，提示需要同时安装 dart 插件的时候选择 yes。如果安装 dart 失败，可以先去搜索 dart 插件并安装，然后再安装 flutter 插件。安装成功，重启 android studio ，欢迎页出现 Start a new Flutter project 选项。

创建 flutter 项目

  1. 选择 Flutter Application 创建项目，其余两个为 Plugin 和 Package。
  2. 设置项目名称、项目路径、项目介绍， 默认 Flutter SDK 路径无误无需更改。
  3. 设置包名，点击 “Finish”。
