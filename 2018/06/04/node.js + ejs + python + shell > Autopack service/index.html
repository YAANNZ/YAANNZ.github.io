node.js + ejs + python + shell > Autopack service

Posted on 2018-06-04 | In iOS

初衷

　　 需求驱动开发，由于混合开发项目，前端同事需要不停的打包测试，为了避免成为专职打包员，所以考虑做一个自动打包服务，前端同事可以访问我的网站提交代码，然后下载打好的安装包。

目标

  • 可以上传代码压缩包
  • 可以自动打包
  • 可以下载安装包

用到的东西

  • node.js （express）
  • ejs
  • python（可以不用）
  • shell
