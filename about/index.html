关于我

　　 一枚 iOS 开发工程师，性别男，90后。多年 iOS 开发经验，作为主力工程师参与设计并开发过多个项目，负责项目架构设计、核心模块的开发、持续集成、自动化部署、App Store上架等任务，浅显的理解 runtime、 内存管理、 多线程、 runloop、 http/https 在 iOS 开发中的使用。有 Cordova、AppCan 平台的混合开发经验，有移动 OA、内容分享、移动 BI、视频直播等类型产品的开发经验，正在寻找一份新工作…

项目经验

同伴-与你轻松办公的同伴

  • 一款移动 OA 产品，累计拥有过万用户量，基于 Cordova 的混合开发方式，主要有移动考勤、审批、工时、邮箱、即时通讯、微应用等模块。
  • 使用 MVC 架构方式开发项目，同时以自定义分类和自定义继承于 NSObject 的工具类的方式来减轻 C 的压力，提高代码的可读性和规范性以便于扩展和维护。
  • 作为主要工程师设计以及开发移动考勤模块，企业创建管理模块，通讯录模块等功能。
  • 使用 FMDB 设计数据持久化方案，同时利用 node.js、python、shell 实现项目的持续集成与自动化部署，提高测试和发布效率。

工程管理-企业版

  • 一款中电基建部门下属全国各地基建工程管理人员使用的移动端产品，基于 Cordova 的混合开发方式。
  • 单独负责项目的开发，实现了基于 CocoaHttpServer 的 App 内启动 HTTP 服务与其他设备局域网通信的功能。
  • 基于 NSURLSessionDataTask 实现的文件下载，支持断点续传下载功能。
  • 基于 AVFoundation.framework Photos.framework 自定义相机，系统相册的创建，相册照片的添加、读取、移动、删除等操作功能。
  • 基于 UIGraphics 类、UIBezierPath 类实现的照片涂鸦、裁剪和添加水印功能。

读图荟

  • 一款内容分享平台的 App，支持用户发帖、评论、分享、关注、举报等功能。

开源项目

  • 一个 Chrome 插件：ChromePlugin
